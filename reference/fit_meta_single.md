# Fit Meta-Analysis for a Single Voxel/ROI

Fit Meta-Analysis for a Single Voxel/ROI

## Usage

``` r
fit_meta_single(y, se, X, method, robust, weights, weights_custom)
```

## Arguments

- y:

  Effect sizes

- se:

  Standard errors

- X:

  Design matrix

- method:

  Meta-analysis method

- robust:

  Robust estimation method

- weights:

  Weighting scheme

- weights_custom:

  Custom weights

## Value

List with meta-analysis results
