# Benchmark fMRI datasets

A list of simulated datasets used for testing the package.

## Format

A list with elements `BM_Canonical_HighSNR`, `BM_Canonical_LowSNR`,
`BM_HRF_Variability_AcrossVoxels`, `BM_Trial_Amplitude_Variability`,
`BM_Complex_Realistic`, and `metadata`. Each element contains simulated
BOLD data and ground truth information.
