# Read jobs exported by export_jobs()

Read jobs exported by export_jobs()

## Usage

``` r
read_jobs(dir)
```

## Arguments

- dir:

  Directory containing `manifest.rds`.

## Value

The list of
[fmri_job](https://bbuchsbaum.github.io/fmrireg/reference/fmri_job.md)s.
