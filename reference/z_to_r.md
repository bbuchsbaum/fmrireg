# Back-transform Fisher's Z to Correlation

Back-transform Fisher's Z to Correlation

## Usage

``` r
z_to_r(z)
```

## Arguments

- z:

  Numeric vector or matrix; Fisher's Z values

## Value

Numeric vector or matrix; correlations

## See also

[`fmri_meta`](https://bbuchsbaum.github.io/fmrireg/reference/fmri_meta.md)

## Examples

``` r
z_to_r(0.2)
#> [1] 0.1973753
```
